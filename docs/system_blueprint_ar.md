# نظام إدارة العمل الأكاديمي اليومي (Academic Intelligence → Research Output)

## الهدف العام
تحويل المتابعة اليومية من **رصد معلومات** إلى **إنتاج مخرجات** (Abstract / Draft / Submission) عبر دورة ثابتة:

**رصد → فرز → تقييم → قرار → تنفيذ → مراجعة**

## مجالات التركيز
- العلوم السياسية المقارنة
- الحوكمة والسياسات العامة
- الديمقراطية/الاستبداد والأنظمة الهجينة
- النخب والسلطة في الملكيات العربية
- الاقتصاد السياسي (خصوصًا الريعي)
- الخليج والشرق الأوسط
- مناهج البحث (QCA, Process Tracing, Qualitative + Quant where needed)

## قاعدة القرار لكل فرصة
لكل عنصر جديد يجب تسجيل:
- `Importance`: High / Medium / Low
- `PhDRelevance`: Very High / High / Medium / Low
- `Decision`: Apply / Track / Ignore / Prepare later

## المخطط التشغيلي
### يوميًا
1. التقاط التحديثات الصباحية.
2. استخراج 3 عناصر فقط:
   - أهم فرصة
   - أهم CFP
   - أهم خطوة تنفيذية
3. اتخاذ قرار واضح لكل عنصر.
4. تنفيذ خطوة بحثية صغيرة (عنوان/فقرة/مراجعة).

### أسبوعيًا
1. مراجعة كامل العناصر الملتقطة.
2. إزالة الضوضاء والتكرار.
3. اختيار فرصة واحدة مركزية للأسبوع.
4. قياس التقدم الفعلي: كتابة/تقديم/متابعة فقط.

### شهريًا
1. تقييم المخرجات: كم abstract/draft/submission؟
2. تقييم التحويل: كم فرصة تحولت إلى فعل؟
3. ضبط المصادر والتركيز وفق النتائج.

## مؤشرات الأداء الرئيسية (KPIs)
1. **Output Indicator**: عدد Abstracts / Drafts / Submissions.
2. **Opportunity Conversion Indicator**: نسبة التحويل من Track إلى Apply/Submit.
3. **Strategic Alignment Indicator**: مدى ارتباط التنفيذ بموضوع الأطروحة.

## الفجوة المركزية الحالية
> الاختناق ليس في الرصد، بل في التنفيذ.

لذلك الأولوية التشغيلية هي: تثبيت دورة كتابة وتسليم قصيرة، وليس توسيع قائمة المصادر.
